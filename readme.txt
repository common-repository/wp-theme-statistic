== Description ==

Wp theme statistic plugin reflect theme statistic data from wordpress.org repository using shortcode. Wp theme statistic plugin shows version no, last theme update date, released date, total download, within days, average download per day, active installation, yesterday download, required WordPress version, required php version and ratings. “ Wp theme statistic “ plugin is useful for theme developer who wants to show their theme download statistic to their viewers in their own site.

== Installation ==

1. Downlad file `wp-theme-statistic.zip`.
2. Upload file to the `/wp-content/plugins/` directory.
3. Activate the plugin through the `Plugins` menu in WordPress.
4. Or you can directly install from wordpress.org
5. After installation , please mention this shortcode [theme-info slug = ‘yourtheme-slug
‘ ] For example : [theme-info slug =‘new-blog’] in any post / page / text widget. So that download statistic of your theme ‘new-blog’ can been seen.
